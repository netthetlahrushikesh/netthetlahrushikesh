## Current Flagship Project

### Used Car Price Intelligence Platform

A trusted-source used car price intelligence project built from data collection to model validation.

**What it includes:**

- trusted used-car dataset preparation
- EDA and data quality checks
- baseline and improved ML models
- model interpretation and segment analysis
- Kaggle dataset packaging
- API-ready model artifact planning
- future website for live price estimation

**Model result:** 10%-class price prediction baseline  
**Primary split MAPE:** 9.88%  
**Repeated split mean MAPE:** 10.33%
